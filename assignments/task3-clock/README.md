# Task 3 - Interactive Clock (Battery + Alarm)

## 기능 요구사항(FR)
- FR1: 배터리 100% 시작, 1초마다 1% 감소
- FR2: 배터리 0% 시 시간 미표시 + 화면 검은색
- FR3: 시/분/초 알람 추가 및 현황 업데이트 (최대 3개)
- FR4(추가기능): 알람 발생 시 Snooze(5분) 기능 제공
