간단 계산기

숫자 2개 + 연산자 1개로 1회 산술 연산을 수행합니다.

결과 -

이벤트 설명

  • submit: 폼 제출 시 계산 수행(기본 새로고침 방지)
  • click: 초기화 버튼 클릭 시 입력값/결과 초기화

함수 설명

  • parseNumber(value): 입력값을 숫자로 변환하고 유효성 검사
  • calculate(a, op, b): 연산자에 따른 산술 연산 수행(0 나누기 예외 처리)
  • render(message, result): 메시지/결과를 화면에 출력

제한 사항

  • 여러 단계 누적 연산은 지원하지 않음(1회 연산만 수행)
  • 수식 파싱(괄호 포함) 및 메모리 기능은 미구현
