# Task 4 – Final Project

## 프로젝트 개요
신입사원 OJT 최종 프로젝트로,
To Do List와 회원가입 기능을 포함한
간단한 웹 애플리케이션을 구현했습니다.

## 주요 기능
### To Do List
- 일정 추가
- 일정 삭제

### 회원가입
- 아이디 중복 체크
- 비밀번호 규칙 검증(8자 이상, 영문/숫자/특수문자)

## 사용 이벤트
- submit: 폼 제출 처리
- input: 실시간 검증
- click: 일정 삭제

## 설계 포인트
- State 객체를 통해 데이터 중앙 관리
- DocumentFragment를 사용한 DOM 렌더링 최적화
- 이벤트 기반 UI 동작

## 제한 사항
- DB 미사용
- 로그인 기능 미구현
