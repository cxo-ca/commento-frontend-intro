# Task 4 – Final Project (To Do List + 회원가입)

## 프로젝트 개요
신입사원 OJT 4주차 최종 프로젝트로,  
To Do List 기능과 회원가입 기능을 포함한 간단한 웹 애플리케이션을 구현했습니다.

## 주요 기능
### To Do List
- 일정 추가
- 일정 삭제

### 회원가입
- 배열 기반 사용자 저장(DB 미사용)
- 아이디 중복 체크(실시간)
- 비밀번호 규칙 검증(8자 이상, 영문/숫자/특수문자 포함)

## 기술 스택
- HTML5
- CSS3
- JavaScript (ES6)
