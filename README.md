# commento-frontend-intro

코멘토 프론트엔드 직무부트캠프  
[1차] 개발 환경 구축 과제용 Repository입니다.

## 과제 내용
- GitHub Repository 생성
- 간단한 자기소개 웹 페이지 구현
- 개발 환경 및 버전 관리 도구 사용 경험

## 소개
- 이름: 정호영
- 전공: 정보통신공학과
- 관심 분야: 데이터사이언스, 데이터 기반 웹 서비스

## 사용 기술
- HTML / CSS / JavaScript
- Git / GitHub

## 실행 방법
1. Repository를 clone 하거나 파일을 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행합니다.

(선택) GitHub Pages를 통해 웹 페이지로도 확인할 수 있습니다.
