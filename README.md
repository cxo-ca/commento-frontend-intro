# Commento Frontend Bootcamp – Assignments

본 저장소는 Commento 프론트엔드 직무 부트캠프 과제를 수행하며 작성한 결과물을 정리한 레포지토리입니다.  
1주차(개발 환경 구축)와 2주차(웹 화면 레이아웃 구성) 과제를 중심으로 구성되어 있습니다.

---

## 1주차 과제 – 개발 환경 구축
- GitHub Repository 생성 및 관리
- HTML / CSS / JavaScript 파일 분리
- 간단한 자기소개 페이지 작성
- GitHub Pages를 통한 결과물 배포

🔗 결과물  
https://cxo-ca.github.io/commento-frontend-intro/

---

## 2주차 과제 – 웹 화면 레이아웃 구성
- 기능 구현 없이 UI 레이아웃 구성
- Grid / Flex 레이아웃 활용
- 선택 주제: **자판기 화면 구성**

🔗 결과물  
https://cxo-ca.github.io/commento-frontend-intro/assignments/task2-vending/

---

## 2주차 선택 과제 (optional2)
HTML/CSS 기초 개념을 문서 및 예제 코드로 정리하였습니다.

- display 속성
- 레이아웃 방식 (Float / Flex / Grid)
- 박스 모델
- class / id 및 CSS Selector
- 자기소개 HTML 페이지 작성

🔗 코드  
https://github.com/cxo-ca/commento-frontend-intro/tree/main/assignments/optional2

---

## 정리
- 1주차: 개발 환경 구축 및 Pages 배포
- 2주차: 레이아웃 중심 화면 구성
- 선택 과제: HTML/CSS 기초 개념 정리
